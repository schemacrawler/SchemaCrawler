--- Connection Options ---

-host=<host>
	Host name. Optional, defaults to localhost.
	
-port=<port>
	Port number. Optional, defaults to 50000.	

-database=<database>
	Database name.
	
-user=<user>
	Database user name.

-password=<password>
	Database password.

-urlx=<properties>
  Additional properties for the JDBC driver.
  Optional, no default.
  For example,
  -urlx=applicationName=SchemaCrawler;readOnly=true
