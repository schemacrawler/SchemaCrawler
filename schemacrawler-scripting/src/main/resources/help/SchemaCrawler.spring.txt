SchemaCrawler - Spring Integration

--- Optional Configuration Options ---

-c <context-file> (short for -context-file <context-file>)
    Reads Spring context XML file from <config-file> instead
    of the default schemacrawler.context.xml

-e <executable> (short for -executable <executable>)
    Uses a bean named <executable> for the executable, instead of the default, a 
    bean named 'executable'

-d <datasource> (short for -datasource <datasource>)
    Uses a bean named <datasource> for a datasource, instead of the default, a 
    bean named 'datasource'
