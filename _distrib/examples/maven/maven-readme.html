SchemaCrawler Maven Plug-in

 

The SchemaCrawler Maven plug-in example demonstrates the use of SchemaCrawler to generate a Maven site report.

 

How to Run

  1. Install maven, and make sure that ant is on your path.
  2. Copy the SchemaCrawler jar file and schemacrawler.config.properties into the maven example directory.
  3. Copy the hsqldb jar file from dbserver directory into the maven example directory.
  4. Start the hsqldb server by running runServer.cmd (runServer.sh on Unix) from the dbserver directory.
  5. Install the SchemaCrawler Maven plug-in by running install-schemacrawler-maven-plugin.cmd (install-schemacrawler-maven-plugin.sh on Unix).
  6. Verify the SchemaCrawler Maven plug-in installation by running verify-schemacrawler-maven-plugin.cmd (verify-schemacrawler-maven-plugin.sh on Unix).
  7. Run mvn site.
  8. See the output produced in the target/site directory.

 

How to Experiment

  1. Try modifying the schemacrawler.config.properties with regular expressions, logging, and so on.
  2. Try using different SchemaCrawler commands in the Maven test project’s pom.xml file. (For a complete list of SchemaCrawler commands, run java –jar <schemacrawler-jar>.)
  3. Try defining a new database connection, using your favorite database and driver.
