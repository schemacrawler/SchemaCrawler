
Available ER Model Loaders:
  attributesloader            Loader for ER model notes, such as remarks or
                                tags and implicit associations
  implicitassociationsloader  Loader for implicit associations in ER Model
  primarymodelloader          Loader for ER Model

Notes:
- For help on an individual ER model loader,
  run SchemaCrawler with options like: `-h loader:implicitassociationsloader`
- Options for the ER model loaders should be provided with the `load` command
