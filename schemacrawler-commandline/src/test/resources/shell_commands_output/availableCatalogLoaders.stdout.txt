
Available SchemaCrawler Catalog Loaders:
  countsloader          Loader for table row counts
  offlineloader         Loader for offline databases
  primarycatalogloader  Loader for SchemaCrawler metadata catalog

Notes:
- For help on an individual catalog loader,
  run SchemaCrawler with options like: `-h loader:implicitassociationsloader`
- Options for the catalog loaders should be provided with the `load` command
